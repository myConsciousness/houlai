# カタログ定義書

## カタログ概要

- **_論理カタログ名:_** リソース種別
- **_物理カタログ名:_** ResourceCategory

## 補足

管理するリソースの種別を定義したカタログです。

## 項目定義

|     | 要素名       | 定義名      | コード | タグ | 説明                 |
| :-: | :----------- | :---------- | :----: | :--: | -------------------- |
|  1  | ドキュメント | DOCUMENT    |   0    |      | ドキュメントリソース |
|  2  | ソースコード | SOURCE_CODE |   1    |      | ソースリソース       |
|  3  | 画像         | IMAGE       |   2    |      | 画像リソース         |
