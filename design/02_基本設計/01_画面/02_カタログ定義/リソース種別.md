# カタログ定義書

## カタログ概要

- **_論理カタログ名:_** リソース種別
- **_物理カタログ名:_** ResourceCategory

## 補足

管理するリソースの種別を定義したカタログです。

## 項目定義

|     | 要素名       | 定義名   | コード | タグ | 説明                             |
| :-: | :----------- | :------- | :----: | :--: | -------------------------------- |
|  1  | ドキュメント | DOCUMENT |   0    |      | ドキュメントリソースを表します。 |
|  2  | ソース       | SOURCE   |   1    |      | ソースリソースを表します。       |
|  3  | 画像         | IMAGE    |   2    |      | 画像リソースを表します。         |
